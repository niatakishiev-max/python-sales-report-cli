# Python Sales Report CLI

Консольная программа для учёта продаж и формирования простых бизнес-отчётов.

## Возможности

- Добавление продажи
- Просмотр списка продаж
- Подсчёт общей выручки
- Подсчёт общего количества проданных товаров
- Отчёт по товарам с группировкой
- Удаление продажи
- Сохранение продаж в JSON
- Загрузка продаж после перезапуска программы
- Проверка неправильного ввода цены и количества
- Защита от пустого названия товара

## Бизнес-логика

Одна продажа хранится как словарь:

```python
{
    "product": "хлеб",
    "price": 100.0,
    "quantity": 2
}
```

Список продаж хранится в `sales.json`.

Программа умеет считать:

- выручку по каждой продаже;
- общую выручку;
- общее количество проданных товаров;
- отчёт по каждому товару.

## Запуск

```bash
python sales_report.py
```
